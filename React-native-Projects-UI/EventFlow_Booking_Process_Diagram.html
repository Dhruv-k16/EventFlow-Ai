Vendor Booking Process Flow Diagram
START: Navigate to /bookings/new
↓
Phase 1: Vendor Selection
🔍
Search & Filter
Browse vendors by category, search by name/service
🎯
Filter Results
Apply category filters (Catering, Decor, Venue, AV/Tech, Photography, Flowers)
👆
Select Vendor
Click on vendor card to select (shows checkmark indicator)
⚠️ Validation Check
Vendor Selected?
If YES ✓
Continue to Event Details
If NO ✗
Show error: "Please select a vendor"
↓
Phase 2: Event Details
📋
Choose Event
Select from existing events or enter new event details
📅
Set Event Date
Pick the date for your event using date picker
📍
Enter Location
Specify venue location with map pin icon
⚠️ Validation Check
Event, Date, Location filled?
If YES ✓
Continue to Service Details
If NO ✗
Show error: "Please fill all event details"
↓
Phase 3: Service Details
📅
Service Date
When the vendor service is needed
⏱️
Duration
How many hours the service is required (e.g., 8 hours)
👥
Guest Count
Number of attendees for capacity planning
📝
Requirements
Special requests or preferences (optional field)
⚠️ Validation Check
Service Date, Duration, Guests filled?
If YES ✓
Continue to Budget & Review
If NO ✗
Show error: "Please fill all service details"
↓
Phase 4: Budget & Review
💰
Enter Budget
Your budget for this vendor service (compared with vendor's price range)
✅
Review Summary
Verify all booking details in organized cards (vendor, event, location, service)
📤
Submit Request
Send booking request to vendor with all provided information
⚠️ Validation Check
Budget entered?
If YES ✓
Submit booking request
If NO ✗
Show error: "Please enter budget"
↓
Successful Booking Flow
👍 Success
Booking request sent to vendor
✅ Redirect
Navigate to Bookings List page (/bookings)
↓
END: Booking request created & user redirected
Key Features & UX Highlights
🎯 Step-by-Step Navigation
Progressive disclosure with Back/Next buttons and clear progress indicator
✅ Real-time Validation
Instant error messages with toast notifications preventing invalid submissions
🔄 Visual Progress
Progress indicator with checkmarks for completed steps, color-coded states
🔍 Smart Search
Filter vendors by name, category, or service type with real-time results
📋 Booking Summary
Complete review before submission with all details organized in cards
✨ Smooth Animations
60fps transitions with Motion (Framer Motion) for professional feel
📱 Responsive Design
Works seamlessly on mobile, tablet, and desktop devices
💜 Brand Consistency
Purple gradient theme (#6E3482 to #A56ABD) throughout the flow
🎨 Visual Feedback
Hover effects, selections, and state changes for better UX
Technical Implementation
State Management
  • currentStep - Track active step (1-4)
  • selectedVendor - Store vendor selection
  • formData - All booking form fields
  • searchQuery - Vendor search filter
  • selectedCategory - Category filter
Key Functions
  • handleNext() - Validate & advance step
  • handleBack() - Return to previous step
  • handleSubmit() - Submit booking request
  • filteredVendors - Filter vendor list
Libraries Used
  • motion/react - Animations & transitions
  • react-router - Navigation after submission
  • lucide-react - Icons throughout UI
  • sonner - Toast notifications
Data Sources
  • MOCK_VENDORS - 6 sample vendors
  • EVENTS - 3 sample events
  • Categories - 7 vendor categories
  • Form Fields - 8 input fields total
